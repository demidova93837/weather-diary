# Weather Diary

A GUI application for tracking weather records with filtering and JSON storage.

## Features
- Add weather entries (date, temperature, conditions).
- Filter records by date or temperature.
- Save/load data to/from JSON file.
- User-friendly interface.

## Technologies
- Python
- Tkinter (for GUI)
- JSON for data storage

## How to run
1. Clone the repository: `git clone [ссылка на репозиторий]`
2. Install dependencies (if any).
3. Run the script: `python weather_diary.py`
